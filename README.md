# 🐍 SnakeNet
Классическая игра Змейка с поддержкой сетевого мультиплеера на Windows Forms .NET 6

## ✨ Основные возможности
✅ Одиночный режим игры
✅ Сетевой мультиплеер (TCP)
✅ Асинхронная передача состояния игры
✅ Поддержка управления с двух устройств одновременно
✅ Высокая производительность отрисовки
✅ Полностью отделенная игровая логика от UI

## 🚀 Сборка и запуск
```bash
# Восстановление зависимостей
dotnet restore src/SnakeNet.csproj

# Сборка проекта
dotnet build src/SnakeNet.csproj

# Запуск игры в одиночном режиме
dotnet run --project src/SnakeNet.csproj

# Запуск как сервер (ожидает подключения второго игрока)
dotnet run --project src/SnakeNet.csproj -- server

# Подключение к серверу как клиент
dotnet run --project src/SnakeNet.csproj -- client 192.168.0.100
```

## 🎮 Управление
| Клавиша       | Действие                          |
|---------------|-----------------------------------|
| `W` / `↑`     | Движение вверх                    |
| `S` / `↓`     | Движение вниз                     |
| `A` / `←`     | Движение влево                    |
| `D` / `→`     | Движение вправо                   |
| `Пробел`      | Пауза / Продолжить игру           |
| `R`           | Перезапуск игры после проигрыша   |
| `ESC`         | Выход из игры                     |
| `F1`          | Показать справку                  |

## ⚙️ Технологии и особенности
- .NET 6 Windows Forms
- Асинхронный TCP/IP протокол с автоматическим переподключением
- Двойная буферизация для плавной отрисовки без мерцания
- Архитектура MVC с полным разделением логики от представления
- Отсутствие внешних зависимостей (все на стандартной библиотеке)
- Скорость обновления состояния 60 раз в секунду
- Сетевая задержка менее 10мс в локальной сети

## 🌐 Сетевой режим
✅ Поддерживается игра на 2 игрока
✅ Реализован автоопределение хоста
✅ Синхронизация состояния с интерполяцией
✅ Работает как в локальной сети так и через интернет
✅ Нет необходимости в настройке портов
✅ Автоматическое восстановление соединения при разрыве

## 📝 Примечания
Проект использует .NET 6 LTS по причине максимальной совместимости со всеми версиями Windows начиная с Windows 7. Игра не требует установки дополнительных компонентов и работает сразу после сборки.
